========================================
INSTALLATION STEPS:
========================================

1. DOWNLOAD THE ZIP:
   - File: multimian-studio.zip
   - Located in your project root

2. UPLOAD TO WORDPRESS:
   - Go to WordPress Admin Dashboard
   - Navigate to: Appearance > Themes
   - Click "Add New" > "Upload Theme"
   - Choose: multimian-studio.zip
   - Click "Install Now"
   - Click "Activate"

3. CONFIGURE THEME:
   - Go to: Appearance > Customize
   - Add Social Media links
   - Add Contact Information
   - Upload logo (optional)

4. CREATE PAGES:
   - Create pages for: Home, Services, Portfolio, About, Contact
   - For Login: Create page, select "Login Page" template
   - For Signup: Create page, select "Signup Page" template

5. SET HOME PAGE:
   - Go to: Settings > Reading
   - Select "A static page"
   - Choose your home page
