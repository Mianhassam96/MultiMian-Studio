## 3. BRAND VOICE & TONE

### Core Voice Attributes

**MultiMian always sounds:**
‚úÖ Professional
‚úÖ Clear
‚úÖ Confident
‚úÖ Solution-focused
‚úÖ Approachable
‚úÖ Results-driven

**Never:**
‚ùå Technical or geeky
‚ùå Overly casual
‚ùå Salesy or pushy
‚ùå Vague or unclear
‚ùå Arrogant

### Language Rules

**DO SAY:**
- Fast, scalable, modern, reliable
- High-performance, secure, user-friendly
- Conversion-focused, results-driven
- Professional, powerful, seamless
- Efficient, optimized, responsive

**DON'T SAY (to clients):**
- React, Node.js, MongoDB
- API endpoints, webhooks
- Component architecture
- State management
- Technical jargon

**Exception:** Only use technical terms when:
- Client specifically asks
- Discussing with technical stakeholders
- In developer documentation

### Tone by Context

**Website Copy:**
- Confident and professional
- Benefit-focused
- Clear and direct
- Persuasive but not pushy

**Client Communication:**
- Friendly and approachable
- Clear and transparent
- Responsive and helpful
- Professional but warm

**Proposals:**
- Professional and authoritative
- Solution-focused
- Data-driven
- Confident

**Social Media:**
- Engaging and informative
- Professional but personable
- Value-driven
- Conversational

## 4. VISUAL IDENTITY SYSTEM

### Color Palette

**Primary Colors:**
- **Navy Blue:** #0B1C2D (Dark backgrounds, headers)
- **Electric Blue:** #2563EB (CTAs, links, accents)
- **White:** #FFFFFF (Backgrounds, text on dark)

**Secondary Colors:**
- **Light Gray:** #F3F4F6 (Backgrounds, sections)
- **Dark Gray:** #1F2937 (Body text)
- **Purple:** #7C3AED (Gradients, accents)

**Accent Colors:**
- **Orange:** #F97316 (Special CTAs, highlights)
- **Green:** #10B981 (Success states)
- **Red:** #EF4444 (Errors, urgent)

### Typography

**Headings:**
- **Font:** Poppins (Primary) / Inter (Alternative)
- **Weights:** Bold (700), SemiBold (600)
- **Sizes:** 
  - H1: 48-72px
  - H2: 36-48px
  - H3: 24-32px
  - H4: 20-24px

**Body Text:**
- **Font:** Inter (Primary) / Open Sans (Alternative)
- **Weight:** Regular (400), Medium (500)
- **Size:** 16-18px
- **Line Height:** 1.6-1.8

**Special Text:**
- **Logo:** Rajdhani (Bold, All Caps)
- **Code/Technical:** Fira Code / Monaco

### Design Principles

**Layout:**
- Clean spacing (80px sections, 24-48px elements)
- Card-based design
- Generous white space
- Clear visual hierarchy
- Professional SaaS-like layout

**Components:**
- Rounded corners (8-16px)
- Subtle shadows
- Smooth transitions
- Hover effects
- Glass morphism effects

**Imagery:**
- Professional photography
- Modern illustrations
- Blue/purple gradient overlays
- Consistent style
