# Images Folder

## Required Images

### Founder Image
**Filename:** `mian-hassam-founder.jpg`
**Location:** `/public/images/mian-hassam-founder.jpg`
**Used on:** About page
**Recommended size:** 800x800px or larger (square aspect ratio)
**Description:** Professional photo of Mian Hassam (founder) with neon/cyberpunk aesthetic

**To add the image:**
1. Save the founder image as `mian-hassam-founder.jpg`
2. Place it in this folder (`public/images/`)
3. The About page will automatically display it

---

## Portfolio Images (Optional)

For portfolio projects, add images with these naming conventions:

- `portfolio/skyscope.jpg` - MultiMian SkyScope Weather project
- `portfolio/ecommerce.jpg` - E-commerce platform project
- `portfolio/dashboard.jpg` - SaaS dashboard project

**Recommended size:** 1200x800px (3:2 aspect ratio)

---

## Testimonial Images (Optional)

For client testimonials, add images with these naming conventions:

- `testimonials/[client-name].jpg`

**Recommended size:** 200x200px (square, will be displayed as circle)

---

## Other Images

You can add any other images needed for:
- Blog posts
- Case studies
- Team photos
- Office photos
- Client logos

Just reference them in your code as `/images/[filename]`
