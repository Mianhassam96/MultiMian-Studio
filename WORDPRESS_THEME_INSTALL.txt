========================================
INSTALLATION STEPS:
========================================

1. UPLOAD THEME:
   - Go to WordPress Admin Dashboard
   - Navigate to: Appearance > Themes
   - Click "Add New" > "Upload Theme"
   - Choose the file: multimian-studio.zip
   - Click "Install Now"
   - Click "Activate" after installation

2. CONFIGURE THEME:
   - Go to: Appearance > Customize
   - Set up Social Media links (Facebook, LinkedIn, GitHub, WhatsApp)
   - Add Contact Information (Phone, Email, Address)
   - Upload your logo (optional)

3. CREATE PAGES:
   - Create pages for: Home, Services, Portfolio, About, Contact, etc.
   - For Login page: Create new page, select "Login Page" template
   - For Signup page: Create new page, select "Signup Page" template

4. SET HOME PAGE:
   - Go to: Settings > Reading
   - Select "A static page" for homepage
   - Choose your home page from dropdown

5. CONFIGURE MENUS:
   - Go to: Appearance > Menus
   - Create "Primary Menu" and add pages
   - Assign to "Primary Menu" location

========================================
CUSTOMIZATION:
========================================

1. COLORS:
   - Edit style.css CSS variables in :root section
   - Primary: #2563eb (Blue)
   - Secondary: #8b5cf6 (Purple)

2. SOCIAL MEDIA:
   - Go to: Appearance > Customize > Social Media
   - Add your social media URLs

3. CONTACT INFO:
   - Go to: Appearance > Customize > Contact Information
   - Add phone, email, and address

4. WIDGETS:
   - Go to: Appearance > Widgets
   - Add widgets to Footer 1, Footer 2, Footer 3 areas

========================================
TROUBLESHOOTING:
========================================

ISSUE: "The theme is missing the style.css stylesheet"
SOLUTION: Make sure you're uploading multimian-studio.zip (not wordpress-theme.zip)

ISSUE: Styles not loading
SOLUTION: Clear WordPress cache and browser cache

ISSUE: AJAX forms not working
SOLUTION: Make sure WordPress jQuery is loaded

ISSUE: Images not showing
SOLUTION: Check file permissions and upload images to Media Library
